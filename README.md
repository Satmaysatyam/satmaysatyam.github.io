# Satyam Gupta Portfolio Site

This is a free, editable GitHub Pages–ready website.

## What is inside
- `index.html` — the full website in one file
- Built-in sections: Home, Research, Publications, Audio Bank, Photography, Art, CV and Anti-CV, Contact, Giving Back
- BeastBox-inspired Audio Bank with upload support for your own recordings

## How to edit
1. Open `index.html` in VS Code or any code editor.
2. Search for the `site = { ... }` object in the `<script>` section.
3. Edit the text, publications, research tabs, images, and contact links.
4. To replace demo sounds with real recordings, upload your audio files in the Audio Bank section.
5. Save and refresh the browser.

## GitHub Pages deployment
1. Create a GitHub repository.
2. Upload `index.html`.
3. In repository settings, enable GitHub Pages from the main branch.
4. GitHub will publish your site automatically.
